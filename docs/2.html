Metrika Consultores Financial Analysis System

Project Description

Designed and implemented a financial web application that serves as the technological backbone of a new financial consultancy firm in México. The consultancy firm is nowdays in process to register as a company and start operating by early 2017. The main service of the consultancy is risk management for financial firms in México called "SOFOMES" which give credit to other companies of strategic economic sectors to improve regional growth.

Project Details

  • Python Programming Language
  • Django Framework
  • MySqlite Database
  • Agile Development
